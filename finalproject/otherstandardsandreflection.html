Reflection

This project was fairly interesting, albeit very challenging. I found it quite difficult to go back to my other labs and attempt to integrate them properly into this whole portfolio. Very tedious if I'm being honest, but it also showed be the importance of being consistent; the web standards and conventional structure of website helped make it easier for me to reference my old work and place it here.

Overall, I think that I learned a great deal during this semester regarding Web Development and Web Standards as a whole. While I can certainly admit that I was not as vigilant or motivated as I should have been, I truly did enjoy the vast majority of the learning experience present within this course. Regardless of my failures and lack of solid effort to match my very forgiving professor, I feel that I was still able to learn a lot throughout the course.

Some particular challenges for me during all of my assignments was finding a way to match the basic HTML structure and continue to expand upon it each week with the Lab Assignments. Sometimes I would get confused and mess thing up, so I would often have to re-read the Learning Module content multiple times.

Despite this all, I truly enjoyed the course, struggle and all, and think that it has expanded my knowledge on how the Web works, as well as the effort put into designing the sites I use every day.
